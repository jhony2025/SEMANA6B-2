# SEMANA6B-2
##Listas Enlazadas y Arreglos en C#
 Descripción del proyecto

Este repositorio contiene la implementación de estructuras de datos dinámicas, específicamente listas enlazadas simples, desarrolladas en lenguaje C#.
El proyecto forma parte de las actividades académicas de la asignatura de programación y tiene como objetivo reforzar la comprensión y aplicación práctica de las listas enlazadas, comparándolas indirectamente con los arreglos tradicionales.
Contenidos desarrollados

El proyecto incluye:

Implementación de la estructura Nodo

Implementación de una Lista Simple Enlazada

Resolución de ejercicios propuestos usando listas enlazadas:

Ejercicio 8: Separación de datos según el promedio

Ejercicio 9: Comparación de dos listas enlazadas (tamaño y contenido)
Estructura del repositorio

Program.cs → Punto de entrada del programa

Nodo.cs → Definición del nodo (dato y referencia)

ListaSimple.cs → Implementación de la lista enlazada simple

Ejercicio8.cs → Lógica del ejercicio 8

Ejercicio9.cs → Lógica del ejercicio 9

README.md → Documentación del proyecto

Ejecución del proyecto

Abrir la carpeta del proyecto en Visual Studio Code

Asegurarse de tener instalado .NET SDK

Ejecutar el comando:

dotnet run


El programa solicitará el ingreso de datos por consola según el ejercicio ejecutado.

El ejercicio que se ejecuta se define en el archivo Program.cs.
Objetivo académico

Comprender el funcionamiento de las listas enlazadas

Aplicar operaciones básicas: inserción, recorrido y comparación

Analizar ventajas frente a los arreglos tradicionales

Fortalecer la lógica de programación y el manejo de estructuras dinámicas
